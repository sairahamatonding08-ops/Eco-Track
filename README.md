# EcoTrack - SDG 12 Web App

A Next.js web application supporting **SDG 12: Responsible Consumption and Production**.

## Features
1. **Carbon Calculator** - Track environmental impact
2. **Eco Marketplace** - Find sustainable products
3. **Community Challenges** - Join sustainability challenges

## Tech Stack
- Next.js 14
- React.js
- Tailwind CSS
- TypeScript
